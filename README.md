简单本地音乐播放页面

说明
- 将本目录作为静态文件目录打开（例如用 VS Code Live Server、或者把文件放到静态服务器下）。
- 页面会加载同一目录下列出的 .m4a 文件（文件名在 `app.js` 顶部的 `tracks` 数组中）。
- 功能：播放/暂停、上一首/下一首、播放进度、播放列表循环开关。

注意事项
- 现代浏览器对自动播放有策略限制：首次播放可能需要用户与页面交互（点击播放按钮或任意位置）。
- 如果你的文件不和页面在同一目录，请修改 `app.js` 中的路径为相对或绝对可访问路径。

如何启动（两种推荐方式）

1) 使用 VS Code + Live Server 插件：
- 打开本文件夹，右键 `index.html` -> "Open with Live Server"。

2) 使用一个简单的本地静态服务器（在 PowerShell 中运行）：
```powershell
# 如果你有 python3
python -m http.server 5500;
# 然后在浏览器打开 http://127.0.0.1:5500/
```
